Integration handshakes

Phase-to-phase contracts (same fields as the console detail drawer).

Handshake: Content → Media

Owner: Media Platform Alliance

Validations:

  • Metadata transfer completeness
  • Format & integrity validation
  • API contract adherence

Evidence: Epic CPTR‑68587 · Integration Sheet (example)

Risks / blockers:

  • Schema drift between CP and MP
  • Partial validation without evidence

Handshake: Media → Data

Owner: Data Alliance

Validations:

  • Ava entity / avail ID availability
  • Entity merge & reconciliation
  • Ratings & localization propagation checks

Evidence: Epic CPTR‑68587

Risks / blockers:

  • Lower-env flow enablement required by upstream teams
  • Avoid reliance on DMC media IDs

Handshake: Data → UI Localization

Owner: UI Localization Alliance

Validations:

  • Content metadata transfer for localization
  • Regional availability rules validation
  • Language-specific content routing

Evidence: Epic CPTR‑68587

Risks / blockers:

  • Metadata completeness for localization workflows
  • Regional content availability conflicts

Handshake: UI Localization → Streaming

Owner: Streaming / Client QA

Validations:

  • Localized content delivery validation
  • Language switching functionality
  • Regional content discovery

Evidence: Epic CPTR‑68587

Risks / blockers:

  • Localized content caching issues
  • Language preference persistence

Handshake: Data → Streaming

Owner: Streaming / Client QA

Validations:

  • Catalog/API response consistency
  • Search & discovery correctness
  • Experience parity checks

Evidence: Epic CPTR‑68587

Risks / blockers:

  • Delayed propagation or caching can mask issues
  • Downstream observability gaps

Migration readiness context

Content migration depends on five critical phases validating handoffs, data integrity, and quality through the pipeline.

Integration dependencies

  • Content → Media: Metadata transfer & format validation
  • Media → Data: Catalog sync & entity reconciliation
  • Data → UI Localization: Regional metadata & content routing
  • UI Localization → Streaming: Language delivery & discovery

Ready indicators

  • Contract validation complete
  • Data flow tested end-to-end
  • Handoff ownership confirmed

Risk indicators

  • Missing integration evidence
  • Unvalidated phase boundaries
  • Incomplete test coverage
